# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in DSoftStudio Mediator, please report it privately. **Do NOT open a public issue.**

Instead, contact the maintainers directly with details of the vulnerability. When reporting, please provide:

- A description of the issue
- Steps to reproduce the vulnerability
- Potential impact
- Suggested mitigation, if known

## Responsible Disclosure

Please allow the maintainers a reasonable amount of time to investigate and resolve the issue before any public disclosure. We are committed to addressing security concerns promptly and will keep you informed of progress toward a fix.

## Supported Versions

The latest released version of DSoftStudio Mediator is actively maintained and receives security updates. Older versions may not receive fixes. We recommend always using the most recent release.
